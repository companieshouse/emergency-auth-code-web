Request an authentication code to be sent to a home address

Use this service to get a company's authentication code posted to an active officer's home address instead of the registered office.

This interim service has been created in response to the coronavirus (COVID-19) outbreak.

To reduce the risk of fraud, we only send authentication codes by post.

You'll need:

  • the company number
  • to sign in to or create a Companies House account, which is different from a WebFiling account

You cannot use this service if the company:

  • has filed a document online in the last 30 days
  • is not based in the UK
  • is not a limited company or limited liability partnership (LLP)
  • is dissolved

How long delivery takes

It takes around 5 working days for an officer to receive an authentication code at their home address.

Due to the impact of coronavirus (COVID-19), it may take longer than usual for post to be delivered.

! Warning You should only use this service if you know the officer has given us their most recent home address. For security reasons, you cannot see or update the address in this service.
