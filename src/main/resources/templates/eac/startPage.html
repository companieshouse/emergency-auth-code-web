Request an authentication code to be sent to a home address

Use this service to get a company's authentication code posted to an officer's home address instead of the registered office.

You should check with company agents to see if they already have the authentication code.

You'll need:

  • the company number
  • to sign in to or create a Companies House account, which is different to a WebFiling account

You cannot use this service if:

  • the company is dissolved
  • the officer is disqualified
  • the officer is bankrupt
  • the officer is on the secure register

! Warning You should only use this service if the officer has previously given us up-to-date details of their home address.

It takes around 10 minutes to request an authentication code.
