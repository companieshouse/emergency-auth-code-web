You cannot use this service

We cannot post an authentication code to an officer's home address.

This may be because:

  • in the last 3 days, someone requested an authentication code for the company
  • in the last 24 hours, you have used the same email address to request 3 authentication codes for different companies

or the company:

  • has filed a document online in the last 30 days
  • does not have any active officers
  • is not based in the UK
  • is a limited partnership (LP)
  • is no longer active (insolvent)
  • only has corporate officers

You are still able to request an authorisation code to the company's registered office address in WebFiling.
